My Professional Journey

It was a time when the covid-19 pandemic was spreading all over the world like a wildfire. I was studying at my 7th Semester. Many companies were laying off their employees and getting a job placement was a big thing. There was a country wide lockdown and I had ample amount of time so at that time Coursera was giving free courses with certifications to the students. I was overwhelmed with this opportunity and jumped on the courses which I liked. I had knowledge on various subjects before enrolling to the course but still I wanted to explore more and I enrolled in courses like HTML,CSS,JS for web developers, problem solving using JAVA, PYTHON programming & AWS. After learning all these I became more confident all thanks to great & generous people of coursera. With my sheer determination I practised for my interview and landed a job in an MNC in first try also it was my first company that I applied for.

I had a training for about one & half month and I learned actually what I needed to learn as a JAVA developer. I learned various backend frameworks of JAVA such as HIBERNATE/JPA, SPRING & SPRING BOOT. As I was getting trained as a full stack java developer I also learned frontend Framework such as ANGULAR. I learned about various DevOps tools such as GIT, Jenkins & JUnit as well. I am a curious person and always have been so when I learned about all these new technologies I felt like I got an Oasis in the desert. I tried to implement practically what I learned till now. So, I started working on the projects. Since the training was time bounded and I was taught limited things but it didn't stop me from learning more. I learned more and more about these technologies everyday as I was doing my projects. I learned futher more from watching youtube videos, learning tutorials and documentations from the internet.

Right now, I have desgined this portfolio page using Angular and I am designing my other projects using it. In upcoming times, I am trying to learn more new technologies such as React, React Native. It is trendy nowadays and I feel cool when I use web Apps and mobile Apps which were developed using these technologies. I am currently working as Senior Software Engineer at LTI (Larsen & Toubro Infotech).
